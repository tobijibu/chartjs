Interactions

hover設定は名前空間options.hoverに渡されます。 グローバルなhover設定はChart.defaults.global.hoverです。 どのイベントがチャートのインタラクションを発生させるかは、 イベントを参照してください。

名称 type 初期値 説明
mode string 'nearest' ツールチップに表示される要素を指定します。 詳細はモードを参照してください。
intersect boolean true trueにした場合、マウスの位置がチャート内の要素に重なった時に適用されます。
axis string 'x' x、y、xyを指定することができ、 距離の計算方向を指定します。 初期値はindexモードの場合はx、 datasetモード、nearestモードの場合はxyです。
animationDuration number 400 hover時のスタイルを変更する場合のアニメーションの時間(ms)。
