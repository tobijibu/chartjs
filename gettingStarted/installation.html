Installation

更新日: 2018.09.05

Chart.jsはnpmかbowerを使ってインストールできます。 インストールはこの方法がおすすめです。

npm

https://www.npmjs.com/package/chart.js

$ npm install chart.js --save

Bower

https://libraries.io/bower/chartjs

$ bower install chart.js --save

CDN

CDNJS

ビルド済みのChart.jsはCDNJSで取得できます。
https://cdnjs.com/libraries/Chart.js

jsDelivr

jsDelivrでも取得できます。
https://www.jsdelivr.com/package/npm/chart.js?path=dist

GitHub

https://github.com/chartjs/Chart.js/releases/latest

最新バージョンをGitHubからダウンロードすることができます。

GitHubからダウンロードした場合や、リポジトリをクローンした場合は、 distファイルを生成するために、ビルドしなければなりません。 Chart.jsにはビルド済みのファイルは付属していないので、 他の方法を強くお勧めします。

正しいビルドを選択する

Chart.jsには2つの異なるビルドが用意されています。

Stand-Aloneバージョン

Files:

  • dist/Chart.js
  • dist/Chart.min.js

このバージョンはChart.jsだけが含まれています。 このバージョンを使っている時に、チャート内で「時間軸」が必要になった場合、 Chart.jsの前にMoment.jsを読み込む必要があります。


Bundlleバージョン

Files:

  • dist/Chart.bundle.js
  • dist/Chart.bundle.min.js

このバージョンにはMoment.jsが含まれています。 1つだけのファイルで「時間軸」を使うには、このバージョンを使う必要があります。 既にMoment.jsを使用している場合は、このバージョンを使用しないでください。 もしMoment.jsを2回呼び出した場合、ロード時間の増加に繋がり、潜在的な問題が発生する可能性があります。
